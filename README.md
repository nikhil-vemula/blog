# Blog

Powered by [Hugo](https://gohugo.io/) static site generator.

Clone with submodules

```bash
git clone --recursive
```

Add post

```bash
hugo new posts/<post>.md
```

Deploy

```bash
./deploy.sh
```

Commit and push the submodule change

```bash
git commit -m "Site updated" --no-verify
git push --no-verify
```
